# Бэклог

## Скиллы и субагенты

### Маппинг задач Битрикс24 → локальные проекты
**Статус**: отложено
**Контекст**: проверили гипотезу — матч возможен по тегам, ключевым словам и описаниям. Основной сигнал — теги задач. Пользователь будет тегировать задачи названием проекта на диске (например, `st`, `seo_text_analyse`).

**Что сделать**:
- Создать файл маппинга (тег/ключевое слово → папка проекта)
- Скилл или субагент, который по задаче находит связанный проект и показывает коммиты/изменения
- Использовать в `daily-review` и `weekly-review` для связки «задача ↔ что реально сделано в коде»

**Выявленные сигналы для матча**:
1. Теги задач (самый надёжный)
2. Ключевые слова в названии
3. Описание задачи
4. Коммиты в проектах (обратный матч)
